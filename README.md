#PROYECTO SOSTENIBLE CON REACT
---

React: Es una biblioteca de JavaScript de código abierto diseñada para construir interfaces de usuario interactivas y eficientes. Basada en una arquitectura de componentes reutilizables, React facilita el desarrollo de aplicaciones web complejas, especialmente las de una sola página (SPAs), permitiendo que la vista se actualice de forma ágil y reactiva ante los cambios de datos.

Node.js: Es un entorno de ejecución multiplataforma que permite ejecutar código JavaScript en el lado del servidor (fuera del navegador). Construido sobre el motor V8 de Google Chrome, Node.js destaca por su modelo de entrada/salida sin bloqueo y orientado a eventos, haciéndolo ideal para desarrollar aplicaciones y APIs rápidas, altamente escalables y eficientes.

Vite: Es una herramienta de construcción (build tool) para el desarrollo frontend de próxima generación que ofrece una experiencia de desarrollo ultrarrápida. Vite proporciona un servidor local que arranca casi al instante y un reemplazo de módulos en caliente (HMR) rapidísimo, optimizando el tiempo de construcción y mejorando significativamente el flujo de trabajo del desarrollador en comparación con los empaquetadores tradicionales.
